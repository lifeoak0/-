皮肤护理指南（Personal Skin Care Guide）
青春痘 是医学上面的 寻常痤疮   （痤疮发生在15-25岁， 雄激素上升 - 皮脂分泌旺盛）
痤疮不是只属于青春期的病，而是一个可以跨年龄的慢性皮肤病
我目前的是 中度炎症性痤疮

核心致病机制
1.皮脂分泌旺盛
2.毛囊口角化异常
3.痤疮丙酸杆菌
4.炎症反应 + 挤压

诱因
1.作息 和 压力  （熬夜 —— 雄激素 —— 爆痘）
2.饮食 高糖 奶茶、甜食，高GI 白米，面包， 乳制品
3.护肤 过度清洁，不用保湿
4.头发遮挡

干预方案
早上使用氨基酸洗面奶、BPO过氧化苯甲胺、 保湿  （使用完BPO以后等5-10分钟，再进行保湿用大宝SOD蜜就可以）
晚上 洗脸 阿达帕林（Adapalene）、保湿

等控制住炎症以后
可以使用壬二酸、烟酰胺


注意事项
不要挤痘，枕巾一周换2-3次
头发不要盖住额头
少油炸、少糖


BPO过氧化苯甲胺 作用核心 杀菌+抗炎，杀灭痤疮丙酸杆菌、减轻红肿、脓包
Adapalene阿达帕林 （维A酸类药物）作用 调节角质代谢、减少粉刺形成、预防新痘

使用美白精华流程
如果要使用美白精华就是晚上使用
现在是先控制痘痘，再提美白
流程是先 洁面 —— 阿达帕林 —— 美白精华 —— 保湿SOD
使用完阿达帕林 等 5-10分钟使用美白精华， 等 1-2分钟使用保湿SOD

生活习惯
每周换1-2两次枕套
不要老乱摸脸
不要老亲Doro
